# Hujan-di-Hari-Kamis
Kinetic Visual Novel, Indonesian Language.
Bercerita tentang seorang anak SMA bernama Alvi yang suatu hari karena suatu alasan ia pulang lebih telat daripada siswa lainnya, tetapi saat ia benar benar akan pulang tiba tiba hujan turun deras disertai petir. Ia memutuskan meneduh sebentar, tetapi ternyata ada orang lain juga yang ikut meneduh.
